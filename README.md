# 📝 Todo List Web App

A modern and responsive Todo List application built using HTML, CSS, and Vanilla JavaScript.  
The app allows users to manage their daily tasks efficiently with a clean UI and smooth user experience.

---

## ✨ Features

- Add, edit and delete tasks
- Mark tasks as completed
- Priority levels (High, Medium, Low)
- Date selection for each task
- Live search functionality
- Filter by:
  - All tasks
  - Active tasks
  - Completed tasks
- Sorting options:
  - Newest first
  - Oldest first
  - High priority
  - Low priority
- Dark Mode / Light Mode
- Local Storage support (data persistence)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
